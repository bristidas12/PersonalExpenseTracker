# PersonalExpenseTracker
A Java-based Personal Expense Tracker that helps users record and manage daily expenses. The system stores expense details like amount, category, and date, displays all records, generates monthly expense reports, identifies the highest expense category, and saves data using file handling.
# Features
• Add and store daily expenses
• Display all recorded expenses
• Generate monthly expense reports
• Identify highest expense category
• Save and load data using file handling
• Test with multiple expense entries
# Technologies Used
• Java
• OOP Concepts
• ArrayList
• HashMap
• File Handling
• LocalDate (Date & Time API)
